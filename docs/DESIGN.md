# DESIGN

## UI方針

License Baseは資格学習全般の上位ブランドとして、信頼感・継続性・情報整理を優先する。

## 方向性

- 白基調
- 青、ネイビー、淡いシアン中心
- 角丸カード
- 薄い境界線
- 控えめな影
- 学習進捗を視覚化
- 問題演習画面は集中しやすく、情報密度はやや高め
- 管理画面は編集効率と視認性を優先

## レイアウト

### 公開ページ

- 上部にブランド説明
- 中央に学習導線
- 資格・Lab単位のカード
- 料金ページへの導線

### 学習ページ

- 単元カード
- 問題カード
- 選択肢リスト
- 解説パネル
- 進捗サマリー
- MVP段階の学習ユーザー識別は、入力パネルとして明示する
- 進捗保存ありの提出と公開採点のみの提出を、文言とバッジで区別する
- 手動の見直し追加は、問題カード右上の小さな操作として配置する

### 学習ダッシュボード

- 上部に学習ユーザー入力と再読み込み操作を配置する
- 進捗サマリーは、回答数・正解数・正答率の3指標をメトリックカードで表示する
- 単元別進捗は、単元名、回答数、正解数、正答率を1行カードで表示する
- 見直し一覧は、不正解由来と手動追加をバッジで区別する
- 権限とプランは、MVP段階では確認用カードとして同一画面に置く
- PC幅は2カラム、スマホ幅は1カラムで縦積みにする

### 管理ページ

- 一覧テーブル
- 検索・フィルタ
- 公開状態バッジ
- 出典確認状態バッジ
- 編集フォーム
