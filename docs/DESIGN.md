# DESIGN

## UI方針

License Baseは資格学習全般の上位ブランドとして、信頼感・継続性・情報整理を優先する。

## 方向性

- 白基調
- 青、ネイビー、淡いシアン中心
- 角丸カード
- 薄い境界線
- 控えめな影
- 学習進捗を視覚化
- 問題演習画面は集中しやすく、情報密度はやや高め
- 管理画面は編集効率と視認性を優先

## レイアウト

### 公開ページ

- 上部にブランド説明
- 中央に学習導線
- 資格・Lab単位のカード
- 料金ページへの導線

### 学習ページ

- 単元カード
- 問題カード
- 選択肢リスト
- 解説パネル
- 進捗サマリー

### 管理ページ

- 一覧テーブル
- 検索・フィルタ
- 公開状態バッジ
- 出典確認状態バッジ
- 編集フォーム

## コンポーネント

- Button
- Card
- Badge
- Input
- Select
- Tabs
- QuestionCard
- ChoiceList
- ExplanationPanel
- ProgressSummary
- AdminTable
- QuestionEditor
