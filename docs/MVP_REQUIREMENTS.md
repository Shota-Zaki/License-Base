# MVP_REQUIREMENTS

## 目的

License Base の最初のMVPとして、Engineer-License-Lab 配下の FE Practice Lab をWebで提供する。

## MVPに含める

- License Baseトップ
- Engineer-License-Labトップ
- FE Practice Labトップ
- 単元一覧
- 無料サンプル問題
- 問題演習フロー
- 解説表示
- ゲスト閲覧
- ログイン後の進捗保存
- 見直し・ブックマーク
- Free / Basic の課金権限土台
- 最小限の管理画面設計

## MVPに含めない

- 全資格領域の実装
- Expoアプリ公開
- App Store / Google Play公開
- 複雑なSNS機能
- AI自動解説生成
- 複雑なゲーミフィケーション

## 初期ユーザー体験

1. ゲストがFE Practice Labの概要を見る。
2. 単元一覧を見る。
3. 無料サンプル問題を解く。
4. 解説を見る。
5. 進捗保存・復習を使う場合はログインする。
6. 有料範囲は権限チェックで制御する。

## 成功条件

- FE Practice Labの学習導線がWebで成立する。
- 問題・選択肢・正答・解説・出典状態をDBで管理できる。
- ゲストとログインユーザーの体験差を制御できる。
- 将来、Java / DB / Security / Cloud / Linuxへ横展開できる。
