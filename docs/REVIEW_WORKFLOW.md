# REVIEW_WORKFLOW

## 目的

FE Practice Lab MVPで、復習対象と手動追加した見直しを同じダッシュボードで扱う。

## 一覧

`GET /v1/me/review-items` は2種類の行を返す。

- `sourceType: review_item`: 不正解由来の復習対象
- `sourceType: bookmark`: 手動追加した見直し

## 操作方針

- 復習対象には「3日後」「7日後」「完了」を表示する。
- 3日後・7日後は `dueAt` を更新する。
- 完了はMVP段階では通常一覧から外す内部状態として扱う。
- 手動追加行には「削除」を表示する。
- 復習対象と手動追加行は同じ一覧に出すが、バッジで区別する。
